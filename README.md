## Pre-Requisite

Fill in your database configuration first in the `.env` file

Example:
```sh
DB_USERNAME = // your database username
DB_HOST = localhost
DB_NAME = // your database name
DB_PASSWORD = // password of your database username 
DB_DIALECT = postgres
DB_PORT = 5432
```

## Install

```sh
$ npm install
```

Then, at the command prompt, type
```sh
$ npm run db:migrate
$ npm run db:seed
```

## Usage

```sh
$ npm start
```
